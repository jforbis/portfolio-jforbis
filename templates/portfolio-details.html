Project Info

With this project, we set out to build a web app that could serve two purposes:

  1. Allow the user to get a list of the top 25, most volatile, stocks for the current date and get some current data on that stock and accessible information about the company.
  2. Using our Machine learning model, predict a closing price of the chosen stock on a particular date that the user chooses.

Disclaimer: current state of web app does not give accurate predictions as Heroku limits the memory of your app at the basic user level; due to this we had to lower the amount of Epochs to '1'. Our full model would include upwards of 500 Epochs; this model has a much higher prediction percentage but takes a little over of 3 hours to train.
